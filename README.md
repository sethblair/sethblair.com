# sethblair.com

Personal portfolio site for Seth Blair — Web and Salesforce Developer.

Built with [React](https://react.dev) and [Mantine 8](https://mantine.dev).

## Getting Started

```bash
npm install
npm start
```

Runs the app locally at `http://localhost:3000`.

## Build

```bash
npm run build
```

Outputs a production-ready bundle to the `build/` folder.

## Stack

- **React 18** — UI framework
- **Mantine 8** — Component library and theming
- **Tabler Icons** — Icon set
